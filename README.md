# Med4U

## Overview

Med4U is a healthcare mobile application built with React Native and Expo Router, designed to streamline access to medical services from home. It enables users to consult doctors via video calls, manage personal health information, and receive prescribed medications delivered to their doorstep. The platform focuses on convenience, accessibility, and continuity of care.

## Features

### Authentication

* Mobile number-based login  
* OTP verification for secure access  
* Seamless and secure user onboarding flow  

### User Profile

* Personal details: name, age, gender  
* Health metrics: height, weight, blood group  
* Medical history: existing diseases, allergies  
* Emergency contact information management  

### Healthcare Services

* On-demand video consultations with doctors  
* Option to visit clinics without additional consultation charges if required  
* Digital prescriptions after consultation  
* Medicine delivery directly to the user’s location without extra delivery cost  

### UI/UX

* Clean and intuitive interface  
* Optimized navigation for quick access to services  
* Responsive design for smooth user experience across devices  

## Tech Stack

* React Native  
* Expo Router  
* JavaScript / TypeScript  
* REST APIs (for backend integration)

## Getting Started

### Prerequisites

* Node.js (v16 or higher)  
* npm or yarn  
* Expo CLI installed globally  

### Installation

```bash
git clone https://github.com/your-username/med4u.git
cd med4u
npm install
```

### Run the Application

```bash
npx expo start
```

* Scan the QR code using Expo Go (Android/iOS) or run on an emulator.
