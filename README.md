# 🏥 Med4U App

Med4U is a modern healthcare mobile application built using **React Native + Expo Router**.  
The app is designed to make healthcare services fast, simple, and accessible from home.

---

## 🚀 Features

### 🔐 Authentication
- Mobile Number Login
- OTP Verification
- Secure User Flow

### 👤 User Profile
- Name
- Age
- Gender
- Height
- Weight
- Blood Group
- Existing Diseases
- Allergy Information
- Emergency Contact

### 🩺 Healthcare Services
- Consult Doctors in 15 Minutes
- Medicine Delivery in 20 Minutes
- Book Lab Tests at Home
- Upcoming Appointments
- Recent Visits History

### 📱 UI / UX
- Modern Mobile UI
- Clean Medical Theme
- Smooth Navigation
- Responsive Design

---

## 🛠 Tech Stack

- React Native
- Expo Router
- TypeScript
- JavaScript
- VS Code
